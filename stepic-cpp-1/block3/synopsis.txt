    3.2 Методы
Метод - функция, определённая внутри структуры
Для вызова метода используется точка, метод вызывается для конкретного экземпляра
Существует неявный параметр this, который указывает на текущий экземпляр структуры
Обьявление методов происходит в заголовочных файлов
Определение методов происходит в .cpp
Абстракция, инкапсуляция
Структуры не содержатся в ассемблерном скомпилированном файле - абстракция

    3.3 Конструкторы и деструкторы
Конструктор - метод, позволяющий инициализировать значения полей
Имеет имя - имя структуры и не имеет возвращаемого значения
Конструктор позволяет создать экземпляр структуры с определенными параметрами
Указывается как с параметрами, так и без
В случае, если конструктор не вызван, поля структуры остаются неинициализированными
Списки инициализации - способ задания конструктора

triangle () : x(0), y(0) {}

Инициализация полей в списке инициализации происходит в порядке обьявления полей
Значения по умолчанию указываются в конструкторах
Нужны в случае вызова конструктора без параметров или с их недостаточным количеством

triangle(int x, int y, int z = 0) : x(x), y(y), z(z){}
