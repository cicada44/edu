    3.1 Структуры

    3.2 Методы
Метод - функция, определённая внутри структуры
Для вызова метода используется точка, метод вызывается для конкретного экземпляра
Существует неявный параметр this, который указывает на текущий экземпляр структуры
Обьявление методов происходит в заголовочных файлов
Определение методов происходит в .cpp
Абстракция, инкапсуляция
Структуры не содержатся в ассемблерном скомпилированном файле - абстракция
