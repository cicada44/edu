fstream fstrm; // создает несвязанный поток
fstream fstrm(s); // создает обьект класса fstream и открывает файл по имени s
fstream fstrm(s, e); // открывает файл по имени s в установленном режиме
fstrm.open(s); // открывает файл по имени s и связывает его с потоком fstrm
fstrm.close() // закрывает файл с которым связан поток fstrm
fstrm.is_open() // возвращает значение типа bool, проверяет, открыт ли файл

    8.2.1 Использование обеьктов файловых потоков
Если функция принимает ссылку на поток istream, то в неё также можно передать и ifstream.
Повторное или провальное открытие файла устанавливает флаг failbit.

    8.2.2 Режимы файла
Каждый поток обладает режимом файла.
in, out - ввод и вывод
app - в конец файла перед каждой записью
ate - в конец файла после открытия
trunc - усекает существующий поток при открытии
binary - ввод-вывод в бинарном формате

Единственный способ сохранить данные в файле при записи - явно указать параметр in или app.
