Вектор - коллекция обьектов одинакового типа.
Индексы целочисленные.
<vector> - шаблон класса.
vector<int> - создаёт вектор с элементами типа int.
Некоторые компиляторы могут потребовать синтаксис vector<vector<int> >.
    3.3.1 Определение и инициализация векторов
Стандартная инициализация

vector<string> svec;

Популярные способы инициализации

vector<T> v1;
vector<T> v2(v1);
vector<T> v2 = v1;
vector<T> v3(n, val); // v3 - {val, val, ... , n}
vector<T> v4(n); // N default values objects
vector<T> v5{1,2,3,4,5};
vector<T> v5 = {1,2,3,4,5};

Поддерживает списочную инициализацию.

vector<int> v = {1,2,3};

Создание определенного количества элементов

vector<int> ivec(10, -1); // {-1,-1,-1,-1 ...}

Инициализирующее значение можно пропускать

vector<int> ivec(10); // {0,0,0,0,0,...}

Фигурные скобки - попытка списочной инициализации.

    3.3.2 Добавление элементов в вектор
Применяяется метод push_back(value).

    3.3.3 Другие операции с векторами
Функции работы с векторами похожи на функции работы со строками.
v.empty(); // void check
v1 == v2; // сравнение всех значений
Индексирование работает только на существующих элементах.
