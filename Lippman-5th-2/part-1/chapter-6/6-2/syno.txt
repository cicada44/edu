Когда парамент - ссылка - функция вызывается по ссылке

    6.2.1 Передача аргументов по значению
В С++ обычно используются ссылки в параметрах, вместо указателей.

    6.2.2 Передача аргументов по ссылке
Ссылочные параметры, которые не изменяются в функции, должны быть обьявлены как const.

    6.2.3 Константные параметры и аргументы
Спецификатор const верхнего уровня игнорируется для обьектов.

    6.2.4 Параметры в виде массива
Функция обработки массива не должна выходить за его пределы!

void print(int (&arr)[10]) // при вызове будет работать только массив из 10 эл.

    6.2.5 Функция main(): обработка параметров командной строки

int main(int argc, char **argv){ /* ... */ }
