По своей сути, istream также считается типом данных.

    2.6.1 Определение типа Sales_data

struct book{ 
    string bookNo; 
    unsigned units_sold = 0;
    double revenure = 0.0; 
}                    // как прмиер

При создании обьекта такого класса для него будут доступны параметры.

    2.6.2 Использование класса
В классе Sales_data необходимо определять поля самостоятельно.

    2.6.3 Создание собственных файлов заголовка
Классы могут быть определены и в функциях, но будут ограничены.
Обычно классы выводят в заголовочные файлы.
Для защиты от повторного включения:

#ifndef SALES_DATA_H // если не существует препроцессорной переменной
#define SALES_DATA_H
#include <string>
struct Sales_data {
    /* ... */
};
#endif // конец

Имена препроцессорных переменных не подчиняются правилам ОВ.

void* - неприводимый к константным обьектам
