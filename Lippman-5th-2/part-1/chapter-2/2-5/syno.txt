    2.5.1 Псевдонимы типов
Псевдоним типа - имя, являющееся синонимом другого типа.

typedef double wages; // - wages - синоним для типа double

Новый стандарт:

using SI = Sales_item;
SI item;

    2.5.2 Спецификатор типа auto
auto - спецификатор типа.
Автоматически преобразовывает тип выражения.

int i = 9;
int j = 1;
auto sum = i + j; // - sum имеет тип int

    2.5.3 Спецификатор типа decltype
Выражение *r возвращает &r;

int i = 0, *r = &i;
decltype (i) p;  // - p имеет тип int
decltype (l) *r; // - возвращает &l, не может быть преобразован
