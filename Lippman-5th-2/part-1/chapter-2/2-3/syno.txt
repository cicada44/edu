    2.3.1 Ссылкиint i
Ссылка - не обьект, а только другое название для уже существующего обьекта.
Нельзя определять ссылкку на ссылку.
Ссылку всегда следует инициализировать.
Ссылка может быть связана только с обьектом, а не с литералом.
Ссылка должна быть такого же значения, как и связанный обьект.

    2.3.2 Указатели
Хранимое в нем значение может быть в 4 состояниях:
    -указывать на обьект
    -указывать на область за концом обьекта
    -не связан ни с каким обьектом
    -может быть недопустимо
Можно по-разному инициализировать указатель нулевым значением.

int *ptr = 0;
int *ptr = NULL;
int *ptr = nullptr; // - новый стандарт

Указатели всегда стоит инициализировать.
Любой ненулевой указатель является true.

void *ptr; // - может указывать на обьект любого типа

    2.3.3 Понятие описаний составных типов

int *p1, p2; // p1 - pointer to type int, p2 - obj of typre int

Ссылка не является обьектом, значет на неё не может указывать указатель.
Но указатель является обьектом, значит на него может ссылаться ссылка.

int i = 42;
int* pi = &i;
int *&fpi = pi;
*r = i;
*r = 0;
