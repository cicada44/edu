    2.2.1 Определения переменных
обьект - именованная область памяти, содержащая какое-либо значение.
Инициализация - не присваивание, происходит при создании.
Присваивание убирает прежнее значение и подставляет новое.
Списочная инициализация - инициализация с помоью списка значений.

int i = 0;
int i = {0};
int i{0};
int i(0)

При определении переменной без инициализатора
происходит инициализация по умолчанию.
Таким переменным присваивается значение по умолчанию.
Встроенные типы вне тела функции - значение 0.
В теле функции - неинициализированные.
При не инициализированном определении обьекта класса
поведения его значений определяется классом.
Можно также указать такой класс, в котором не может существовать неиниц. обьектов.
